# LIVO Core – MVP (Login, Register, Dashboard)

This project is a first MVP demo of **LIVO Core**, a real-time AI translation platform.

## Features (MVP)

- Register page (mock)
- Login page (mock)
- Redirect to Dashboard after successful login
- LIVO Core Dashboard UI with:
  - Active devices (mock)
  - Sessions today (mock)
  - Translation minutes (mock)
  - Quick Actions area (future features)
  - Recent Activity preview (mock)

## Tech Stack

- Next.js (App Router)
- TypeScript
- React
- Deployed on Vercel
- Developed in GitHub Codespaces

## Demo URL

Production deployment (Vercel):

`https://livo-ai-wold.vercel.app`
